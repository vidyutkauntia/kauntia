# Kauntia Motor Works

Interactive single-page portfolio for **Kauntia Motor Works Pvt. Ltd.** — building India's capital-light e-bus operating platform.

Static site (HTML/CSS/JS, no build step), hosted free on GitHub Pages.

- `index.html` — the page
- `assets/css/style.css` — styles
- `assets/js/main.js` — interactions
- `assets/img/` — logo + photos
